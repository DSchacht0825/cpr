# Community Property Rescue
## Application Features Overview

---

### About the Platform

Community Property Rescue is a comprehensive housing assistance management platform designed to help organizations support homeowners facing foreclosure. The system streamlines the entire process from initial application through field worker engagement and case resolution.

---

## Public-Facing Features

### Online Application Portal
- Mobile-responsive application form accessible from any device
- Collects essential homeowner information:
  - Contact details (name, phone, email)
  - Property address and location
  - Foreclosure status indicators
  - Auction date tracking
  - Notice of Default (NOD) status
  - Notice of Trustee Sale (NTS) status
- Instant submission confirmation
- Progressive Web App (PWA) capability for home screen installation

---

## Administrative Dashboard

### Application Management
- Real-time view of all submitted applications
- Status tracking system:
  - Pending
  - Contacted
  - In Progress
  - Closed
- Filter applications by:
  - Status
  - ZIP code
  - Custom date ranges
- One-click access to full application details
- Assign cases to field workers directly from dashboard

### Urgent Case Identification
- Automatic flagging of urgent auctions (within 7 days)
- Visual indicators for:
  - Notice of Default
  - Notice of Trustee Sale
  - Scheduled auction dates
- Priority sorting for time-sensitive cases

### Worker Assignment System
- Dropdown assignment of cases to field workers
- Reassignment capability between workers
- Track which worker is handling each case
- View worker's current caseload

---

## Field Worker Mobile App (PWA)

### Secure Worker Portal
- Individual worker login credentials
- Session management with secure authentication
- Mobile-optimized interface for field use

### Worker Dashboard
- View assigned cases with priority indicators
- Quick access to:
  - New field visit logging
  - Field intake forms
  - Visit history
  - Case details

### Field Visit Logging
- **Visit Type Selection:**
  - Initial Contact
  - Follow Up
  - Property Inspection
  - Document Collection

- **Visit Outcome Tracking:**
  - Attempt (no contact made)
  - Engagement (successful contact)

- **GPS Location Capture:**
  - Automatic location recording
  - Verification of field presence

- **Photo Documentation:**
  - Camera integration for property photos
  - Photo type categorization:
    - Exterior
    - Interior
    - Damage
    - Documents
  - Caption support for each photo

- **Detailed Notes:**
  - Property condition observations
  - Occupant situation documentation
  - Immediate needs identification
  - General visit notes

- **Follow-Up Scheduling:**
  - Set follow-up dates
  - Add follow-up notes
  - Auto-scheduling for attempts (3-day default)

### Property/Customer Search
- Search by name, address, phone, or email
- Browse all applications feature
- Quick access to any property in system
- Any worker can log visits on any property

### Field Intake Capability
- Complete new applications in the field
- GPS-tagged submissions
- Capture all required information on-site
- Immediate sync to main system

## Reports & Analytics

### Interactive Heat Map
- Visual display of all field visit locations
- Mapbox-powered interactive map
- Color-coded visit markers:
  - Green: Successful engagements
  - Amber: Attempts (no contact)
- Cluster view for high-density areas
- Click for visit details

### Key Metrics Dashboard
- **Clickable stat cards showing:**
  - Total Applications (click to view all)
  - Total Field Visits (click to view all)
  - Pending Review count (click to view pending)
  - Urgent Auctions (click to view urgent cases)

### Worker Accountability Tracking
- **Attempt vs Engagement metrics:**
  - Total attempts across all workers
  - Total engagements across all workers
  - Overall engagement rate percentage
  - Pending follow-ups count

- **Individual Worker Performance:**
  - Total visits per worker
  - Attempts per worker
  - Engagements per worker
  - Engagement rate by worker
  - Color-coded performance indicators

### No Activity Report
- Identify properties without visits in last 7 days
- Shows unassigned properties
- Direct links to take action
- Helps ensure no case is neglected

### Worker Activity Reports
- Select individual worker
- Filter by custom date range
- View complete visit history
- See all properties worked
- Export individual worker data to CSV

### Data Breakdown Charts
- Applications by Status (visual bar chart)
- Applications by County
- Field Visits by Type
- Field Visits by Worker

### Data Export
- Export all applications to CSV
- Export all field visits to CSV
- Includes all relevant data fields
- Date-stamped file naming
